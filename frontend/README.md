# React + TypeScript Frontend

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Runs at http://localhost:5173

Ensure the backend is running at http://127.0.0.1:8000. The dev server proxies `/api` to the backend.

Optional: Create `.env` with `VITE_API_URL=http://127.0.0.1:8000` to call the backend directly.
